There are 5 games: Check is Even, calcucator, find great common divisor, find missed number in arithmetic progression, is prime number.

Minimum node version:
```
v19.7.0
```

How to install:
```
make install
npm link
```

How to run games:
```
brain-calc
brain-even
brain-gcd
brain-prime
brain-progression
```
